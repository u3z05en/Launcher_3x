Overview

This application is a group based application launcher written using PyQt. The application uses a simple n:m database to store applications and groups and the links between them. Applications duplicated across multiple groups are the same database instance. It has been tested to work on Windows and Linux.

Shortcuts

Ctrl+Click:   Multiple apps for launching.
Shift+Click:  Multiple contiguous apps for launching.
Caps+Caps:    Minimise & Restore the window.
RCtrl+RCtrl:  Same as above.
Esc:          Minimise the window to the tray.
Ctrl+Q:       Exit the application.
